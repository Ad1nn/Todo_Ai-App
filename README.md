# Todo App - Phase 1

In-Memory Console Todo Application for Hackathon II - Evolution of Todo.

## Quick Start

```bash
# Run the application
uv run todo

# Run tests
uv run pytest
```

## Features

- Add tasks with title and description
- View all tasks with status indicators
- Mark tasks as complete/incomplete
- Update task details
- Delete tasks

See `specs/001-console-todo-app/quickstart.md` for detailed usage.
